# ADR-0009: Provider Abstraction with OpenAI as the First Adapter

- Status: Superseded by ADR-0015
- Date: 2026-07-21

## Context

The original foundation selected OpenAI as the first content-generation adapter.

## Decision

This decision is superseded because the product now requires explicit selection among Nano Banana, ChatGPT, and Claude, and because prompt assistance must be separated from image rendering.

See ADR-0015 and ADR-0016.
